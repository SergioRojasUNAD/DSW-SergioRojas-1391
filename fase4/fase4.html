Responda las siguientes preguntas

1. ¿Qué es un selector CSS de selección múltiple?

a. Un selector que permite seleccionar varios elementos con una sola regla.
b. Un selector que solo se aplica a elementos específicos en una página web.
c. Un selector que solo se aplica a elementos que cumplen con ciertos criterios de estilo.

2. ¿Cuál es el operador utilizado en un selector de selección múltiple?

a. +
b. ,
c. ~

3. ¿Qué selectores se pueden combinar en un selector de selección múltiple?

a. Selectores de elemento y de clase.
b. Selectores de elemento y de ID.
c. Cualquier combinación de selectores.

4. ¿Pueden usarse selectores de pseudoclase en un selector de selección múltiple?

a. Sí, pero solo si se combinan con selectores de clase.
b. No, no es posible
c. Sí, se pueden combinar con cualquier tipo de selector.

5. ¿Qué selector se puede usar para seleccionar todos los elementos de un mismo tipo dentro de un contenedor específico?

a. El selector de clase.
b. El selector descendente.
c. El selector ascendente.
